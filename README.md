# The Compiler For AndroSwan Dual Boot

Un outil graphique pour compiler Android et Linux, et configurer un dual boot sur appareils compatibles.

## Fonctionnalités
- Interface graphique (Tkinter)
- Compilation Android et Linux
- Configuration automatique du dual boot
- Installateur Windows via Inno Setup

## Installation
1. Téléchargez `gui.exe` ou compilez avec `gui.py`
2. Lancez l’installateur généré avec `TheCompileForAndroSwan.iss`

## Dépendances
- Python 3.10+
- PyInstaller (pour compiler en .exe)
- Inno Setup (pour créer l’installateur)
